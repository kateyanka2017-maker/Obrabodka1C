 Учебная обработка 1С

Проект содержит решение учебной задачи по программированию в среде 1С:Предприятие.

 Реализованный функционал:
 Поиск и фильтрация данных по ведомостям.
 Условное оформление списка.
 Обработка пользовательского ввода с различными методами вывода информации (сообщения, предупреждения, исключения).
